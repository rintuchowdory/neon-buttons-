# Rintu Chowdory — Portfolio

Personal portfolio website built with **Next.js** and deployed on **Vercel**.

🌐 **Live:** [rintu-portfolio.vercel.app](https://rintu-portfolio.vercel.app)

## Stack

- Next.js + React
- Tailwind CSS
- Cloudflare Workers (API proxy)
- Vercel (hosting)

## Features

- 85+ GitHub projects showcase with filter tabs
- Work experience & CV timeline
- Skills & tech stack overview
- Contact form

## Run locally

```bash
git clone https://github.com/rintuchowdory/my-portfolio
cd my-portfolio
npm install
npm run dev
```
